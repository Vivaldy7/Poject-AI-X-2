# Jarvis

An AI-powered WhatsApp chatbot built for Haitian small businesses, helping automate customer interactions without the need for dedicated support staff.

## Stack

- Node.js + Express
- Google Gemini API
- Twilio (WhatsApp integration)

## Setup

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your API keys (see `.env.example` if provided)
4. Run `node server.js`
